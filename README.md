# Tic-Tac-Toe AI using Minimax Algorithm

A console-based Tic-Tac-Toe game developed in C++ featuring an AI opponent powered by the Minimax algorithm.

## Features

* Human vs AI gameplay
* Unbeatable AI using Minimax
* Win, loss, and draw detection
* Input validation
* Console-based interface

## Technologies

C++, Recursion, Minimax Algorithm, DSA
